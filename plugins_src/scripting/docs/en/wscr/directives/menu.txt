menu Directive
<code><pre>
<b>menu</b> "Color" 'blue' {
  item "Red" 'red'
  item "Green" 'green'
  item "Blue" 'blue'
}
</pre></code>

Shows a drop down menu to select choices.

The list of values each start with <b>item</b>, followed
by the string showed to user, followed by an atom that
will be used by the script.

A <b>menu</b> directive should be inside a
<b>params { ... }</b> section.
